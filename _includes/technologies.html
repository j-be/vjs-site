Technologies

The Jump Simulator is a combintaion of the following technologies

Steel frame

Size: 3m x 3m x 3m
Weight: ~1000kg
Designer: Bela Eckermann
Manufacturer: Waagner Biro

Unity3D

To develop the software for the Jump Simulator we relied on the Unity3D game engine.

Oculus Rift DK2

The Oculus Rift DK2 is a state of the art virtual reality headset.

Kinect v2

The Kinect v2 is a motion sensing input device by Microsoft which was initially developed for their XBox gaming console.

Raspberry Pi

The Raspberry Pi is a single-board computer developed for teaching computer science at schools. Apart from that it is a great little thingy for creating hardware-software projects in no time.

Arduino

Arduinos are a set of small, open-source microcontroller board. It is perfect for taking first steps in microcontroller programming.
